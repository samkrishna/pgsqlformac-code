Query Tool for PostgreSQL

Reload Schema

Select the database again in the task bar.

Create SQL

Select a table, column, view or other item in the schema view and right click (or control-click). A menu will pop-up and give several options for automatically creating SQL and pasting it into the query window.

Connections Preferences

The last connection is kept in user preferences. To change just login to another database. It will be updated automatically.

View Font Preferences

Are also kept in your user preferences. Just change the fonts and they will be saved.

Preferences

Are kept in "~/Library/Preferences/com.druware.postgres.QueryTool.plist".

The Query Tool Website

Is at https://sourceforge.net/projects/pgsqlformac.
